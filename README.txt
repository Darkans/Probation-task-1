Это результат одного из моих тестовых заданий. 
Задание было следующее: дана папка, в которой находится произвольное количество файлов, каждый из которых содержит неизвестное количество строк, состоящие из строки и числа, разделенные символом ','. Строка содержит название города, число - количество жителей. Имя города может использоваться несколько раз. 
Ожидаемый результат: новый файл того же формата, в котором устранены повторения имен городов, а число должно содержать сумму всех его жителей.
